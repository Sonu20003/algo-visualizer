Layer 7

Application Layer

The application layer provides network services directly to end-users. This is where protocols like HTTP, HTTPS, FTP, SMTP, and DNS operate. Your web browser, email client, and other applications interact at this level.

Examples: Web browsers, Email clients, File transfer applications

Layer 6

Presentation Layer

The presentation layer transforms data between the application layer and the network. It handles encryption, decryption, compression, and data formatting. This layer ensures that data sent by one system can be read by another.

Functions: Encryption/Decryption, Compression, Data formatting

Layer 5

Session Layer

The session layer manages communication sessions between applications. It establishes, maintains, and terminates connections between local and remote applications. This layer also handles session checkpointing and recovery.

Functions: Session establishment, Session maintenance, Session termination

Layer 4

Transport Layer

The transport layer provides reliable data transfer services to upper layers. It handles segmentation, flow control, and error recovery. TCP provides reliable connection-oriented service, while UDP provides faster connectionless service.

Protocols: TCP (reliable), UDP (fast), Port numbers for application identification

Layer 3

Network Layer

The network layer handles routing of data packets between different networks. It uses logical addressing (IP addresses) to determine the best path for data delivery across multiple networks. Routers operate at this layer.

Functions: Routing, Logical addressing, Path determination

Layer 1

Physical Layer

The physical layer transmits raw bit streams over physical mediums. It deals with electrical signals, light pulses, or radio waves. This includes cables, fiber optics, wireless signals, and network hardware.
